# Projeto_WEB
🌐 Site Pessoal - Projeto Web
- Sobre o Projeto

Este projeto consiste no desenvolvimento de um site pessoal utilizando HTML puro, criado com o objetivo de apresentar informações sobre mim, minha trajetória acadêmica e alguns dos projetos realizados durante o curso.

O site funciona como um pequeno portfólio acadêmico, reunindo páginas com informações pessoais, interesses e trabalhos desenvolvidos ao longo das disciplinas.

O projeto foi desenvolvido como parte das atividades da disciplina PW III (Programação Web III), sob orientação do professor André Olímpio.

🎯 Objetivo

O principal objetivo deste projeto é praticar os fundamentos do desenvolvimento web, utilizando apenas HTML para estruturar as páginas.

🛠️ Tecnologias Utilizadas

HTML5

Este projeto foi desenvolvido sem o uso de frameworks ou CSS avançado, com foco no aprendizado da estrutura fundamental do HTML.

📩 To-DO List - Projeto Web

Este projeto consiste no desenvolvimento de uma to-do list utilizando HTML, CSS e JavaScript, criado com o objetivo de gerenciar tarefas.

O projeto, funciona como um pequeno gerenciamento de tarefas, consistindo na organização, sendo possivel inserir a tarefa, marcar como concluida, e apagar para poder adicionar outra.

🛠️ Tecnologias Utilizadas

HTML5, CSS, JavaScript

 🛒 E-commerce - Cafeteria Ivy's coffee shop - Projeto Web

Este projeto consiste no desenvolvimento de uma pagina E-commerce simulando uma cafeteria chamada "Ivy's coffee shop", criado com o objetivo de fazer a compra de um produto, e ir direto para o carrinho de compras, assim fazendo a soma, o total de itens comprados. 

Um pequeno E-commerce com a seleção dos produtos, soma de cada produto, a junção de diversos produtos em um carrinho só, ele faz parte do processo de estudos do JavaScript.

🛠️ Tecnologias Utilizadas

HTML5, CSS, JavaScript.
